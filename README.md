# 🎯 Guessing Number Game (Java Swing)

A simple GUI-based number guessing game built using Java Swing. The application generates a random number between **1 and 100**, and the player tries to guess it with helpful hints after each attempt.

---

## 📌 Features

* 🎲 Random number generation (1–100)
* 🖥️ User-friendly GUI using Java Swing
* 📉 Hint if the guess is too low
* 📈 Hint if the guess is too high
* 🎉 Success message on correct guess
* 🔄 Automatic game reset after winning
* ⚠️ Input validation for invalid or out-of-range entries

---

## 🛠️ Technologies Used

* Java
* Swing (GUI Framework)
* NetBeans IDE (GUI Builder)

---

## 🚀 How to Run

1. Clone or download this repository.
2. Open the project in your IDE (recommended: NetBeans).
3. Compile and run the `GuessingNumberGame.java` file.
4. The game window will appear.
5. Enter your guess and click **"Guess ?"**.

---

## 🎮 How to Play

1. Enter a number between **1 and 100**.
2. Click the **Guess** button.
3. The game will respond with:

   * "Too low" 📉
   * "Too high" 📈
   * "Correct!" 🎉
4. Keep guessing until you win!
5. After winning, a new number is automatically generated.

---

## ⚠️ Input Rules

* Only numbers between **1 and 100** are allowed.
* Invalid input (like text or symbols) will show an error message.
